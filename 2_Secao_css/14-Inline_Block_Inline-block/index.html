Elementos de Bloco

  • Inline
  • Block
  • Inline-Block


Elemento Bloco

Vai ocupar todo o espaço disponível na tela

Exemplo:h1 p table

eles se organizam um abaixo do outro



Elemento Inline

ocupa o espaço referente ao seu conteúdo...

Exemplo:a span img

Eles se organizam um ao lado do outro



Elemento Inline-block

ocupa o espaço referente ao seu conteúdo...

Exemplo:a span img

Eles se organizam um abaixo do outro desde que não caiba no espaço disponível
